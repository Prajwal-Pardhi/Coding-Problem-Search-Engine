# Leetcode Question Scrapper

##Scrapping Procedure

1. run lc.py file. (fetch all 'a' tag links containg "/problems/" and save it to lc.txt)
2. run cleaner.py file. (removes the solution links and save remaining links to lc_problems.txt)
3. run problems_explore.py ( Go through each problem link and store its heading and description(folderwise) in Qdata folder)
